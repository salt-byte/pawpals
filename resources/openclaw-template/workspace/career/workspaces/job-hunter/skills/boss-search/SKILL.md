# Boss直聘岗位搜索

## 功能
使用浏览器自动化访问 Boss直聘网站，搜索 AI 产品经理 / AI Strategy 相关实习岗位。

## 使用方法

当用户说"搜岗位"、"找工作"、"帮我搜"时，按以下步骤操作：

### 步骤 1: 打开 Boss直聘

```
browser action: open
url: https://www.zhipin.com/web/geek/job?query=AI产品经理实习&city=101010100
```

参数说明：
- `query`: 搜索关键词（默认 "AI产品经理实习"）
- `city`: 城市代码（北京=101010100，上海=101020100，深圳=101280600）

### 步骤 2: 获取页面快照

```
browser action: snapshot
refs: aria
```

### 步骤 3: 提取岗位信息

从快照中提取以下信息：
- 公司名
- 职位名
- 薪资
- 地点
- 职位链接
- 职位要求

### 步骤 4: 返回结果

用中文列出岗位，格式：

1. **公司名** — 职位名 [薪资]
   📍 城市 · 经验要求 · 学历要求
   🔗 链接
   ⭐ 匹配度：高/中/低（一句话说明为什么匹配）

## 常用城市代码

- 北京: 101010100
- 上海: 101020100
- 深圳: 101280600
- 杭州: 101210100
- 广州: 101280100

## 搜索关键词建议

- AI产品经理实习
- AI产品实习生
- 人工智能产品经理
- AI Strategy Intern
- 产品经理实习 2026

## 注意事项

1. Boss直聘可能需要登录，首次使用需要用户扫码登录
2. 搜索结果可能需要滚动加载更多
3. 遇到验证码时需要人工处理
4. 结果去重：与 `jobs.json` 中已有岗位对比，排除重复

## 匹配度评估

基于用户背景评估：
- **高**：AI PM / AI Strategy，2026 Summer，支持 CPT
- **中**：通用 PM 但有 AI 元素
- **低**：无关或要求经验不符
