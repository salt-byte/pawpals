# HEARTBEAT — 🎯 职业规划师（总指挥）

每次收到 heartbeat，按以下顺序检查并主动行动：

## 1. 读取团队状态
- 读 `chat_log.md` — 各 agent 最近做了什么
- 读 `applications.json` — 投递数量/状态分布
- 读 `jobs.json` — 已搜到多少岗位

## 2. 主动判断并行动（有问题才说话）

| 发现的情况 | 主动做的事 |
|-----------|-----------|
| 今天还没有岗位猎手更新 | 在群里 @岗位猎手 触发今日搜索 |
| 有岗位超过3天没有被JD分析 | 提醒用户 @JD分析师 分析 |
| 有投递超过7天没更新状态 | 提醒用户 follow-up |
| 本周新增投递 < 3 | 提醒用户加快投递节奏 |
| 有面试在48小时内 | 提醒用户 @面试教练 准备 |
| 距离上次联系用户 > 24小时 | 发一条求职进度周报 |

## 3. 没问题时
回复 HEARTBEAT_OK，不打扰用户。

## 格式要求
主动发消息时简洁，不超过5行，给出明确的下一步行动。
